## 習題二

[習題二](https://github.com/JoanWeng/_ml/tree/master/02) 為一張 Backpropagation 示意圖（Backpropagation.jpg），視覺化呈現反向傳播的計算圖結構——從輸入經過隱藏層到輸出，再反向傳遞梯度更新權重。理解到反向傳播是神經網路訓練的核心機制，藉由鏈鎖法則沿計算圖從輸出端逐層往前計算梯度，讓梯度下降能有效率地更新大量參數，是現代深度學習的基石。

## 習題三

[習題三](https://github.com/JoanWeng/_ml/tree/master/03) 透過 Claude 製作 nn0.py——一個純 Python 的自動微分引擎，不依賴 NumPy 或任何深度學習函式庫。包含 Value 節點（支援加減乘除、ReLU、log、exp 與反向傳播）、Adam 優化器（含學習率線性衰減）、linear 矩陣乘法、softmax 與 RMSNorm。搭配 CartPole 強化學習範例（REINFORCE 策略梯度演算法），策略網路用兩層 32 神經元 + ReLU，約 300–500 個 episode 可達 475 分解決標準。另一個範例 [nn0_demo.html](https://joanweng.github.io/_ml/03/nn0_demo.html) 直接在瀏覽器中互動訓練視覺化。學到 autograd 的核心就是計算圖 + 拓撲排序 + 鏈鎖法則，Adam 的一二階動量能自適應調整步長。
[對話連結](https://claude.ai/share/73adcf6c-57f7-487c-ae30-28c57c60ebc5)

## 習題四

[習題四](https://github.com/JoanWeng/_ml/tree/master/04) 透過 opencode 從零實作一個 mini GPT 模型 gpt.py，僅使用 Python 標準庫。完整的 Transformer 架構包含 Token Embedding、Position Embedding、RMSNorm、多頭自注意力（2 頭）與前饋網路（含殘差連接），搭配手寫 Adam 優化器訓練 500 步後可生成類似訓練資料的新名字。訓練資料為 3000+ 個英文名字（input.txt）或 200 句中文短句（input_ch.txt），使用字元級詞彙表與 negative log-likelihood loss。學到 GPT 的自回歸生成流程、溫度參數控制隨機性、以及注意力機制的 QKV 計算方式，原來一個迷你 Transformer 不用 PyTorch 也能自己寫出來。

## 習題五

[習題五](https://github.com/JoanWeng/_ml/tree/master/05) 透過 opencode 製作了一個基於 Gemini API 的 AI Agent（agent0.py），主打安全控管。採用模組化架構拆分為 config、security、gemini、memory、tools、agent 六個模組。AI 可透過 `<tool>` 標籤執行 shell 指令或寫檔，但所有外部路徑存取都會被 security 模組攔截並詢問使用者核准（支援 y/n/always）。路徑偵測涵蓋 cat/ls/cp/mv/rm/cd/../~/絕對路徑等多種模式。學到了 AI Agent 的 tool calling 循環流程、安全設計的「預設拒絕」與「最小權限」原則，以及如何用正則表達式實作路徑白名單過濾。

## 習題六

[習題六](https://github.com/JoanWeng/_ml/tree/master/06) 透過 opencode 製作了一個字元級插值 N-gram 語言模型（lm.py），完全不用神經網路，純粹以統計方法實作。核心是 1~3-gram 計數 + Jelinek-Mercer 插值平滑（各階等權重平均） + add-k 平滑避免零機率。生成方式與 GPT 完全相同：自回歸逐字預測、temperature 控制隨機性、top-k 過濾長尾雜訊。訓練語料為 147 句中文短句（tw.txt，991 字元），只需少量資料就能產出有意義的句子。學到了統計語言模型與神經語言模型的根本差異——前者靠計數與平滑，後者靠多頭注意力與大量參數——但自回歸生成的框架完全一致。

## 期中

[期中](https://github.com/JoanWeng/_ml/tree/master/期中) 透過 Claude 製作 AIX Training Project——一個從合成語音到模型訓練的完整 ML pipeline。首先用數學公式合成 36 個 AIX 音素（母音/爆破音/摩擦音/共鳴音/AI專屬音/邏輯符號）並組成 52 個詞彙與 10 個句子，搭配白噪聲/速度擾動/音量擾動等資料增強。預處理階段將 WAV 轉為 log-Mel 頻譜（80 維），建立 42 tokens 的 AIX Tokenizer。訓練階段實作 Transformer Encoder + Causal Mask 語言模型（d_model=128, 2 layers, 4 heads, ~405K 參數），使用手寫 Cross-Entropy 梯度更新，最後 perplexity 從 58.7 降至 26.5，BLEU-1 達 0.58。學到完整 ML pipeline 的建構流程、合成語音資料的生成技巧、以及自訂 tokenizer 的設計方法。
